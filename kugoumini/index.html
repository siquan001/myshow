Kugou Mini Player

使用方法

  1. 点击左侧图片可显示和关闭歌词，hover可显示控制面板
  2. 点调整声音
  3. 点打开歌曲列表
  4. 点搜索歌曲播放

引用方法

KugouMiniPlayer固定高度50px，请找合适位置安放。

HTML:

<link rel="stylesheet" href="./kugoumini.css">
<kugou-mini id="id"></kugou-mini>
<script src="./kugoumini.js"></script>

JavaScript:

KugouMiniPlayer.init('id',[musicList:Array])

musicList格式

[{
  "hash":"musicHash",// like 02e945cac3e2371685880c8050aeaa72
  "albumid":"musicAlbumId", //like 28378420
  "name":"musicName",// like 艾索 - 晚安喵
  "musicUrl":"musicUrl"// 音乐url ，非必须
},...]
